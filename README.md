# FDS

Factorial Digit Sum

## Overview

Simple function that takes in a number, calculates the factorial of that number, and returns the sum of all the digits of the factorial.

## Tech Used

* Javascript
* Node

## Unit Testing

* Mocha

```bash

npm install
# to run the code
node fds.js
# to run the test
mocha
```
